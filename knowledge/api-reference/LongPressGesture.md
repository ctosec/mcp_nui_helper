# LongPressGesture

**Namespace:** `Tizen.NUI`
**Inherits:** `Gesture`
**Category:** basic

A long press gesture is emitted when the user holds the screen with the stated number of fingers.<br />     A long press gesture finishes when all touches have been released.<br />

## Properties

| Name | Type | Description |
|------|------|-------------|
| NumberOfTouches | `uint` | A long press gesture is emitted when the user holds the screen with the stated number of fingers.<br />     A long press gesture finishes when all touches have been released.<br /> |
| ScreenPoint | `Vector2` | This is the point, in screen coordinates, where the long press occurred.<br />         If a multi-touch long press, then this is the centroid of all the touch points.<br /> |
| LocalPoint | `Vector2` | This is the point, in local actor coordinates, where the long press occurred.<br />         If a multi-touch long press, then this is the centroid of all the touch points.<br /> |
