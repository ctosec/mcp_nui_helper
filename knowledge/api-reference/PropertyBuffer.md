# PropertyBuffer

**Namespace:** `Tizen.NUI`
**Inherits:** `BaseHandle`
**Category:** basic

PropertyBuffer is a handle to an object that contains a buffer of structured data.<br />     PropertyBuffers can be used to provide data to Geometry objects.
