# DrawableGroup

**Namespace:** `Tizen.NUI.BaseComponents.VectorGraphics`
**Inherits:** `Drawable`
**Category:** basic

A class enabling to hold many Drawable objects.     The added Drawables are affected by the transform and opacity of DrawableGoup.

## Methods

| Name | Signature | Description |
|------|-----------|-------------|
| AddDrawable | `void AddDrawable(Drawable drawable)` | A class enabling to hold many Drawable objects.     The added Drawables are affected by the transform and opacity of DrawableGoup. |
| RemoveDrawable | `void RemoveDrawable(Drawable drawable)` | Remove drawable object from the DrawableGroup.         This method is similar to deregistration. |
| RemoveAllDrawables | `bool RemoveAllDrawables()` | Clears the drawable object added to the DrawableGroup.         This method does not free the memory of the added drawable object. |
