# MouseRelative

**Namespace:** `Tizen.NUI`
**Inherits:** `Disposable`
**Category:** basic

MouseRelative is used when relative mouse movement occurs in the window.

| TimeStamp | `uint` | Gets the time the mouse relative evnet is being started |
| DeviceClass | `DeviceClassType` | Gets the device class the mouse relative event originated from. |
