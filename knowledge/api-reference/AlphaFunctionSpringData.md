# AlphaFunctionSpringData

**Namespace:** `Tizen.NUI`
**Category:** basic

Enumeration for predefined spring animation types.     This presets are based on typical spring behavior tuned for different motion effects.

## Properties

| Name | Type | Description |
|------|------|-------------|
| AlphaFunctionSpringData | `class` | Enumeration for predefined spring animation types.     This presets are based on typical spring behavior tuned for different motion effects. |
| Stiffness | `float` | Spring stiffness(Hooke's constant). Higher values make the spring snap back faster. Minimum value is 0.1.         Default Value is 1.0. |
| Damping | `float` | Damping coefficient. Controls oscillation and settling. Minimum value is 0.1.         Default Value is 1.0. |
| Mass | `float` | Mass of the object. Affects inertia and the duration of the motion. Minimum value is 0.1.         Default Value is 1.0. |
