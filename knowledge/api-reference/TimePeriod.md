# TimePeriod

**Namespace:** `Tizen.NUI`
**Inherits:** `Disposable`
**Category:** basic

TimePeriod is used to define delay and duration of a process such as <see cref="Transition"/>.     TimePeriod is composed of Delay and Duration in milliseconds
