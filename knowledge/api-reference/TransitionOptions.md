# TransitionOptions

**Namespace:** `Tizen.NUI`
**Inherits:** `IDisposable`
**Category:** basic

This TransitionOptions class is a class to control Transition motion.     This class includes multiple options for the Transition.     NUI supports various kinds of Transitions such as App transition, Page transition, and so on.

| TransitionTag | `string` | This TransitionOptions class is a class to control Transition motion.     This class includes multiple options for the Transition.     NUI supports various kinds of Transitions such as App transition, Page transition, and so on. |
| TransitionWithChild | `bool` | Property for Page transition.         A View could be transition with its child Views or without them.         Default value is false |
