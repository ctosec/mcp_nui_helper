# WatchTime

**Namespace:** `Tizen.NUI`
**Inherits:** `Disposable`
**Category:** basic

The WatchTime class is used to get time for the WatchApplication.<br/>     A WatchTime has a time handle from watch application framework.<br/>     You can get time(hour, minute, second, millisecond) and date(year, month, day)<br/>     on receiving timeTick signal.
