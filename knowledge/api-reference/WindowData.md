# WindowData

**Namespace:** `Tizen.NUI`
**Inherits:** `Disposable`
**Category:** container

This class represents the default window data for an Application object.     It contains information about the default window.

## Properties

| Name | Type | Description |
|------|------|-------------|
| PositionSize | `Rectangle` | This class represents the default window data for an Application object.     It contains information about the default window. |
| WindowType | `WindowType` | Gets or sets the mode of the default window.         The default mode is WindowMode.Transparent. |
| Screen | `string` | Gets or sets the screen of window data. |
| FrontBufferRendering | `bool` | Gets or sets the front buffer rendering of the default window.         The default value is false. |
| BorderInterface | `IBorderInterface` | Gets or sets the border interface of the default window.         The default value is null. |
